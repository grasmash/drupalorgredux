Built for this moment

Architecture that agents understand

25 years building the world's most scalable, secure CMS — structured content, API-first design, fine-grained access control, and robust configuration management. The foundation AI needs was in Drupal's DNA long before agents arrived.

MCP Protocol

Native Model Context Protocol support. Claude, Cursor, Copilot — any MCP-compatible agent connects and operates your site.

Install: composer require drupal/mcp && drush en mcp

API-First by Design

JSON:API ships in core. Typed content models, semantic configuration, and CLI automation give agents a complete, machine-readable interface — no scraping, no guessing.

Governance Built In

Roles, permissions, and access control at the foundational layer. Scope agent access precisely — "create articles, edit own drafts, can't touch users." Trust by design, not afterthought.

Context Control Center

Brand voice, audiences, and key messages in one place. Every agent — built-in or external — draws from this single source of truth. Your brand stays consistent at scale.

Human-in-the-Loop

Every AI action queued for review. Batch approvals, audit trails, revision history. The open web ethos of transparency and responsible use, applied to AI.

# Install DDEV (local dev environment)
$ curl -fsSL https://ddev.com/install.sh | bash

# Create & launch your Drupal CMS site
$ mkdir my-site && cd my-site
$ ddev config --project-type=drupal --docroot=web
$ ddev composer create-project drupal/cms
$ ddev launch
