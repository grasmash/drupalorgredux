# Verified Drupal Resource URLs

All URLs below have been fetched and confirmed to return valid pages (not 404) as of 2026-03-23.

## 1. Drupal CMS API Docs (JSON:API)
- **URL:** https://www.drupal.org/docs/core-modules-and-themes/core-modules/jsonapi-module
- **Description:** Comprehensive guide to Drupal's JSON:API core module covering installation, configuration, filtering, pagination, sorting, and CRUD operations.

## 2. Drupal Core Repository (GitLab)
- **URL:** https://git.drupalcode.org/project/drupal
- **Description:** The main Drupal core repository on GitLab (git.drupalcode.org). Shows 47,457+ commits, 42 branches, 741 tags.

## 3. Drupal Developer Guide
- **URL:** https://www.drupal.org/docs/develop
- **Description:** Main developer documentation hub covering local setup, tools, Composer, Git, module/theme development, APIs, testing, and migration.

## 4. Drupal CMS Desktop App (Launcher) Download
- **URL:** https://new.drupal.org/drupal-cms/launcher
- **Description:** Official download page for the Drupal CMS Launcher desktop app. Available for macOS, Windows, and Linux. Includes built-in web server, SQLite, and Drush.
- **See also:** https://new.drupal.org/docs/drupal-cms/get-started/cms-launcher (installation docs)

## 5. Drupal Free Trial / Sandbox
- **URL:** https://simplytest.me/
- **Description:** Free service to evaluate Drupal modules, themes, and distributions in temporary sandbox environments without local installation. Login with admin/admin.

## 6. Drupal AI Initiative Page
- **URL:** https://www.drupal.org/about/starshot/initiatives/ai
- **Description:** Official Drupal AI Initiative hub page. Lists leadership, partners, strategy docs, AI Academy, blog posts, and event info.
- **See also:** https://new.drupal.org/ai (marketing-oriented AI landing page promoting "Drupal - The best AI-powered Open Source CMS")

## 7. Drupal AI Initiative Blog Post (Dries Buytaert)
- **URL:** https://dri.es/accelerating-ai-innovation-in-drupal
- **Description:** "Accelerating AI innovation in Drupal" by Dries Buytaert, published June 9, 2025. Announces the Drupal AI Initiative launch with $100K+ in funding.
- **See also:** https://www.drupal.org/blog/drupals-ai-roadmap-for-2026 (2026 roadmap post)

## 8. Drupal Context Control Center (CCC) Module
- **URL:** https://www.drupal.org/project/ai_context
- **Description:** Module for storing, managing, and injecting contextual information (brand voice, style guides, regulatory requirements) for AI agents. Alpha release, supports Drupal 10.3+ and 11.

## 9. Drupal MCP Module
- **URL:** https://www.drupal.org/project/mcp
- **Description:** Model Context Protocol module enabling Drupal to serve as an MCP server. Currently merging with the MCP Server module. 368 sites using it. Stable release 1.2.3.
- **See also:**
  - https://www.drupal.org/project/mcp_server (MCP Server - full protocol support with OAuth 2.1)
  - https://www.drupal.org/project/mcp_client (MCP Client - integrates with Drupal AI module)
  - https://www.drupal.org/project/mcp_tools (MCP Tools - connects to Claude, Cursor, Windsurf)
  - https://www.drupal.org/project/mcp_studio (MCP Studio - no-code MCP tool builder)

## 11. Drupal CMS Download Page
- **URL:** https://new.drupal.org/download
- **Description:** Central download hub for Drupal CMS and Drupal Core, plus extensions (modules, themes, recipes). Offers links to Drupal CMS (for marketers/designers) and Drupal Core (for developers).
- **See also:** https://www.drupal.org/project/cms (Drupal CMS project page, currently at version 2.0.0)

## 12. Drupal Community Page
- **URL:** https://www.drupal.org/community
- **Description:** Community hub describing how to get involved. Covers community groups, contributor guide, Working Group, diversity initiatives, mentoring, and support resources.

## 13. Drupal Security Page
- **URL:** https://www.drupal.org/security
- **Description:** Official security advisories section. Lists vulnerability disclosures for Drupal Core and contributed modules with CVE IDs, risk assessments, and RSS feeds.

## 14. Drupal Contribute Page
- **URL:** https://www.drupal.org/contribute
- **Description:** Guide to 14 contribution areas including accessibility, community building, core development, documentation, event planning, mentoring, translation, and usability.
